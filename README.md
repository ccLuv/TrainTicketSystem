# 火车票务管理系统

## 项目简介
这是一个用C++实现的简单火车票务管理系统，支持管理员和乘客两种角色。

## 功能特点
- 管理员：添加/删除列车、查看所有列车和座位信息
- 乘客：购票、退票、查询列车和车票信息
- 数据持久化存储到本地文件

## 技术栈
- C++
- Visual Studio 2022
- 文件流操作

## 运行方法
1. 使用 Visual Studio 2022 打开 ProjectXX.sln
2. 选择 Release 配置
3. 按 F5 运行或 Ctrl+F5 运行（不调试）

## 使用说明
启动后选择身份：
- 输入 1：管理员模式
- 输入 2：乘客模式

## 注意事项
- 程序会在同目录下自动创建 1.dat 和 2.dat 数据文件
- 建议以管理员身份运行（如需写入权限）
